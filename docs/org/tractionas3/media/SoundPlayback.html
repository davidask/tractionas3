setSound()method 
public function setSound(sound:Sound, resetPosition:Boolean = true):void

Changes the sound of the SoundPlayback. If the Soundplayback is currently playing, the new sound will begin playing automatically. Use SoundPlayback.stop() SoundPlayback.pause() before calling this method if you wish to ensure that the new sound is not playing.

Parameters
sound:Sound — Reference to new Sound.
 
resetPosition:Boolean (default = true) — If true the new sound will begin playing from start. If set to false the new sound will attempt to play at the same position as the old sound.
