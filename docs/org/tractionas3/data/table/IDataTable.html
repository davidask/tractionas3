public function addField(name:String, dataType:Class, primary:Boolean = false):DatatableField

Adds a field to the data table. A field is strictly type bound.

Parameters
name:String — Field name to be assigned as identifier.
 
dataType:Class — Data type for the field. A data type of a field cannot be changed.
 
primary:Boolean (default = false) — Defines whether the field is the primary field of the data table.

Returns
