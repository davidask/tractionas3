Allows you to set marks defining a point in a loading process.

A LoadMonitor instance defines a custom loading progress allowing your application to react to when a certain amount of data has loaded - complete files or partially.

Unlike the LoaderQueue the LoadMonitor does not interfere with loaders, it is simply an indicator. Use LoadMonitor when you e.g. want your applicaiton to respond to when one or several file streams have reached a a certain progress.

Property detail
byteProgressproperty
byteProgress:Number  [read-only]

Indicates the progress of all the loaders references by a LoadMonitor with regard to actual byte size. Note that this method only returns an accurate progress indication if the file size of each loader is known.

Use of the prefetchBytesTotal() method can retrieve a file size before loading it.
