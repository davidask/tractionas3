Packageorg.tractionas3.core
Classpublic class Application
InheritanceApplication CoreSprite flash.display.Sprite

Application provides a main entry point for your application and should be extended for use as document class only. It stores the stage, root and loaderInfo in static properties automatically. Extend Application for modified usage.

To reference the stage, root or loaderInfo from anywhere in your application simply use the Application.stage, Application.root or Application.loaderInfo static properties.

Constructor detail
Application()constructor
public function Application()

Creates a new Application object. An Application instance expects to have access to stage, root and loaderInfo properties on instantiation. Using an instance of Application as other than document class will throw an error.

Application automatically sets the stage scaleMode property to StageScaleMode.NO_SCALE and the align property to StageAlign.TOP_LEFT.
