Property detail
filtersproperty
filters:Array  [read-write]

An indexed array that contains each filter object currently associated with the CoreSprite.

CoreSprite implements a modified filter structure allowing you to add and remove any specified filter. You can explicitly add or remove filters using the adddFilter() or removeFilter() methods. Changing property of a filter requires an updateFilters() method call, however adding or removing a filter does not.
