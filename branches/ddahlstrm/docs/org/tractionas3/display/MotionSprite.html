render()method 
public override function render():void

Renders the RenderableSprite.

By default this method is executed every frame if the MotionSprite is currently in the display list. However, if manually stopped using stopRender(), it will not start rendering automatically when added to the display list after being previously removed.

startRender()method 
public override function startRender():void

Starts rendering of the MotionSprite.

NOTE: By default this method is called when the MotionSprite is added to stage. Use startRender() if you previously called stopRender() and want to re-enable automatic render control.
