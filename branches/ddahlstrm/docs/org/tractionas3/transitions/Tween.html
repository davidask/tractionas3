ImplementsResetable, Runnable

Tween provides an extendable API base for creating value tweens.

Rather than changing property values of an object, Tween is independent and dispatches events indicating the tween progress. Tween outputs on a value between two specified values (start value and end value), and this value can be used to represent a tween.
