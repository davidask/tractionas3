Image Constructor.

Parameters
source:* — The source of the image. Can be of the following types: String, URLRequest, DisplayLoader, BitmapLoader or BitmapDataLoader. If the type is either String or URLRequest a BitmapDataLoader will be created.
 
imgWidth:Number — Image width
 
imgHeight:Number — Image height
 
loadMethod:uint — The load method of the image.
