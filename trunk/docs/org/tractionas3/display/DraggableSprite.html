Packageorg.tractionas3.display
Classpublic class DraggableSprite
InheritanceDraggableSprite MotionSprite RenderableSprite CoreSprite flash.display.Sprite
ImplementsICoreInterface
SubclassesSimpleVisualProfiler, ThrowableSprite

DraggableSprite directly implements drag and drop functionality and responds to mouse events similar to flash.display.Sprite.startDrag() and flash.display.Sprite.stopDrag() methods.

Using startDrag() and stopDrag() methods on DragabbleSprite is not allowed and will throw an error.
