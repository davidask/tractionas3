Classpublic class ClearableEventDispatcher
InheritanceClearableEventDispatcher flash.events.EventDispatcher
ImplementsIClearableEventDispatcher, ICoreInterface
SubclassesWeakEventDispatcher

ClearableEventDispatcher keeps track of what events have been added, giving the posibility to remove all added event listeners via removeEventListeners() method defined in IClearableEventDispatcher interface.
