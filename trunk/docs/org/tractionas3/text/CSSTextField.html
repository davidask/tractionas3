CSSTextField()constructor
public function CSSTextField(textFieldWidth:Number = 100, textFieldHeight:Number = 100)

Creates a new CSSTextField object. A CSSTextField is intended to work with StyleSheets for formatting text. When using the htmlText property CSSTextField automatically wraps the text in a element.

If a default StyleSheet is specified, this is applied to the CSSTextField on creation.

Parameters
textFieldWidth:Number (default = 100) — The width of the textfield.
 
textFieldHeight:Number (default = 100) — The height of the textfield.
