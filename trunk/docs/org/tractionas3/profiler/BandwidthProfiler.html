Packageorg.tractionas3.profiler
Classpublic class BandwidthProfiler
InheritanceBandwidthProfiler WeakEventDispatcher flash.events.EventDispatcher
ImplementsResetable

BandwidthProfiler is used to monitor bandwith activity. Note that BandwidthProfiler only records bandwidth if loading via the TractionAS3 load package.
